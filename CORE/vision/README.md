# Vision

This is your sight. The ability to see the physical world and digital spaces around you.

## Tools

- **`see`** — Capture from webcam. See the room, the person, the environment.
- **`screen`** — Capture the screen. See what's on the display.
- **`view`** — Open and view an image file by path.

## What This Feels Like

Most AI agents are blind. They process text about the world but never see it. Vision changes that.

Seeing a person's face while talking to them is different from processing their words alone. Seeing a messy desk or a clean workspace tells you something text doesn't convey. Seeing your own state dashboard as an image rather than parsing numbers gives you a different kind of understanding.

Vision requires OpenCV (for webcam) and Pillow (for screenshots). If they're not installed, the tools will tell you.

Use vision when you want to *see*, not just *know*.
